Rendering Synthetic Objects into Real Scenes

Nathan Petreaca, Frankie Eder, Jule Ahmed

Our goal is to add CG objects to a photo, including the actual light from the scene. In essence, we hope to recreate the results of “Rendering Synthetic Objects into Real Scenes: Bridging Traditional and Image-based Graphics with Global Illumination and High Dynamic Range Photography“ (1998) by Paul Debevec. To capture the light and shape of the environment, we will use a metal sphere and a well-tested camera calibration method.

Goals and Deliverables

Metrics

The main metric we will use to determine the performance of our project will be the quality of the images we are able to generate. However, we suspect that we will be able to use the response curves from each channel to quantitatively measure our correctness.

Analysis

We want to understand which method(s) yields the best response curves for HDR images, i.e. we will also compare results between standard 0-255 images and HDR images. We also want to build a high-level understanding how HDR and global illumination interact for the application of environment mapping.

Deliverables

Goals

We plan to create .exr environment maps using HDR photos of a metal sphere, and implement them into our Project 3 framework.

Aspirations

If time allows, we hope to be able to add CG objects to a photo using the lighting from the scenes we are capturing. The challenge in this problem mainly lies in mapping the 3D space of the CG objects into the photograph, and we anticipate this will be difficult and time consuming.

Schedule

    4. Tone map images for viewing

Week 3

  • Correct environment maps from mirrored sphere
    1. Create correct mapping scheme from circular coordinates
    2. Implement differential rendering to improve speed

Week 4

  • Putting CG objects into scenes
    1. Gather more test photos
    2. Calibrate scene using calibration grid
    3. Build/collect test 3D Scenes
    4. Map 3D scenes to the 2D image scene using calibrated grid
    5. Corrections
